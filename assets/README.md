# Assets

Save the workshop logo here as **`logo.png`** (this exact name).

- It is shown as the centerpiece of the hero, used as the favicon, and
  used as the social-share (Open Graph) image.
- A transparent-background PNG looks best on the light hero.
- The hero displays it up to 420px wide; export at ~2× (≈840px) for
  crispness on retina screens.

If `logo.png` is missing, the hero automatically falls back to the
text title, so nothing breaks before the file is added.
