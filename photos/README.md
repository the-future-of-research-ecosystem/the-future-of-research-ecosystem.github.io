# Headshots

Drop each person's photo into this folder using the **exact filename**
below. The page loads it automatically — no HTML editing needed. If a
file is missing, the card falls back to the colored initials.

## How to add a photo

1. Save the headshot as `<filename>.jpg` (lowercase, exactly as listed).
2. Square images work best (they are cropped to a circle). ~400×400px is plenty.
3. Refresh the page — the photo appears.

> All filenames end in `.jpg`. If you only have a PNG, either convert it
> to JPG, or change the `src="..."` extension for that one person in
> `index.html`.
